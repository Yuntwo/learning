SELECT 从 FROM 开始
牛客 SQL 快速入门 42 题沉淀 · 6 个常踩的坑
1
执行顺序不是书写顺序
FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY
2
别名在 WHERE 里用不了
GROUP BY / HAVING / ORDER BY 多数库能用
3
SUM(CASE) 忘写 ELSE 会变 NULL
全 NULL → SUM=NULL；换 COUNT(CASE) 更稳
4
LEFT JOIN 从表条件写 ON 不是 WHERE
写 WHERE 会退化成 INNER JOIN
5
每组 TOP-N 用窗口函数
ROW_NUMBER() OVER (PARTITION BY .. ORDER BY ..)
6
列套函数 = 走不了索引
YEAR(col)=2021 改写成范围条件
